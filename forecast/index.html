Zero-friction option: let your agent do it. Paste this to Claude Code or Codex and it bundles your whole history into one file in Downloads; then drop that single file above. Bundle every .jsonl under my ~/.claude/projects into one file, TOP-claude-export.jsonl, in my Downloads folder (not a cloud-synced folder like Desktop, these logs contain my prompts), adding a newline between files so records never merge. Windows PowerShell: $o="$env:USERPROFILE\Downloads\TOP-claude-export.jsonl"; Remove-Item $o -EA 0; Get-ChildItem "$env:USERPROFILE\.claude\projects" -Recurse -Filter *.jsonl | ForEach-Object { Add-Content $o (Get-Content $_.FullName -Raw) }. Mac or Linux: find ~/.claude/projects -name '*.jsonl' -exec awk 1 {} + > ~/Downloads/TOP-claude-export.jsonl Copies only the prompt shown above. One merged file parses identically to the folder; sessions are grouped by their IDs, not by file. If the merged file tops ~800 MB, use the folder picker instead; browsers cannot read single files that large.
