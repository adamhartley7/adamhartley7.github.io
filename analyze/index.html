How Route B works

  1. Copy the script below and save it as strip_claude_logs.py.
  2. On Windows, run py strip_claude_logs.py. On macOS or Linux, run python3 strip_claude_logs.py. It writes claude-usage-only.jsonl next to it.
  3. Open that file and check it yourself. It is content-free: no prompts, replies, code, file contents, or raw session IDs. It can include timestamps, model labels, permission modes, and locally generated session numbers.
  4. Drop that file into the box below to see it parsed locally on your device.
  5. Download a CSV or JSON copy for yourself. TOP is not accepting external files yet.
