Route B · text-free file you build yourself

Maximum privacy. You run a short, readable script on your own machine. It reads your Claude Code logs and writes ONE new file that holds only numbers and labels: model, the four token counts, timestamps, and permission mode (whether you were steering it live or it ran on its own). No prompt text, no replies, no code, no file contents, ever. That text is never copied into the file in the first place. Read the script first, run it, check the file it makes, then decide whether to share it. Your original logs are never touched.

How Route B works

  1. Copy the script below and save it as strip_claude_logs.py.
  2. Run python3 strip_claude_logs.py. It writes claude-usage-only.jsonl next to it.
  3. Open that file and check it yourself: every line is just numbers and labels, no text.
  4. Drop that file into the box below to see it parsed locally on your device.
  5. Press download your text-free data and send Adam the small CSV or JSON.
Nothing here uploads on its own. Route B never auto-sends: you download the file and send it yourself, only if you want to.
strip_claude_logs.py
What is in the file you would send: model names, input / output / cache-read / cache-write token counts, timestamps, session ids, permission mode, and turn counts. What is never in it: your prompts, the assistant's replies, any code, any file contents. You can read both the script and the file it produces before anything is shared.
